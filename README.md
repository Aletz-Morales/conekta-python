Conekta
=========

Wrapper for api.conekta.io

Install

```sh
pip install conekta
```

```sh
easy_install conekta
```


Charge via bank:

```python
import conekta

conekta.api_key = 'blablabla'

customer = conekta.Customers(
    name="Gilberto Gil",
    email="gil.gil@mypayments.mx",
    phone=5567942342,
    street1="Jiminez 11",
    street2="Despacho 99",
    street3="La Condesa",
    city="Cuauhtemoc",
    state="DF",
    country="MX",
    zip="06100"
)

bank = conekta.Banks(type="banorte")

charge = conekta.Charge.create(
    amount=10000,
    currency="MXN",
    description="DVD - Zorro",
    customer=custome,
    bank=bank
)

print charge.parseJSON()

```
